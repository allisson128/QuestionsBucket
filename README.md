- Banco de Questões
- Amigo do Estudante
- Auxílio do Concurseiro

  ** Criar hierarquia com base nas matérias;
* Sistema 

TABELA: QUESTÕES
ID, Enunciado, Fonte, Matéria, Tipo (aberto/Fechado)
